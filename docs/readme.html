$Id$ CONTENTS programs description license files documentation installation examples PROGRAMS ff - "For each File" fl - "For each Line" DESCRIPTION For each File allows users to execute a Bash script recursively upon many files and directories. It is ideal for performing a repetitive procedure such as mass file renaming. For each Line allows users to execute a Bash script, once for each line in many files. COPYING A copy of the GNU GPL version 2 license. DOCUMENTATION 1. To view help message for "For each File", run './ff -h' 2. To view help message for "For each Line", run './fl -h' 3. View program source code; they are Bash scripts! INSTALLATION Run './install' or perform a manual installation as follows. 1. Set the file permissions so that you can execute the programs. Example: chmod u+x ff fl 2. Copy the programs to a location recognized by your ${PATH} variable. Example: cp ff fl ~/bin EXAMPLES I recommend you first invoke the 'ff.vars' and 'fl.vars' examples to see how the scripting variables behave. If you wish to execute one of the provided example scripts, say, on all files in the current working directory, you would run the following. Example: ff * < [path to example file] Example: cat [path to example file] | ff * To execute an example script recursively on the current working directory, run the following. Example: ff -r . < [path to example file] Example: cat [path to example file] | ff -r .
